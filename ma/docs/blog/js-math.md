## Math 对象

Math 对象用于执行数学任务。

## Math 对象属性

| 方法          | 描述          | 举例          |
| ------------- |:-------------:|:-------------:|
| random()      | 返回介于 0 ~ 1 之间的一个随机数 | Math.random() => 0.7030866345513442|
| round()      | 把一个数字舍入为最接近的整数(四舍五入) | Math.round(0.49) => 0;<br> Math.round(0.50) => 1; |
| floor()      | 是向下取整计算。返回值小于等于 x，且与 x 最接近的整数 | Math.floor(0.60) => 0;<br> Math.floor(-5.1) => -6; |
